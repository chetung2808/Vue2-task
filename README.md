

# Development Preparation
1. Open Terminal (inside VSC) (Ctrl + `)
1. Run `docker-compose up -d` (accept sharing resources if prompts)
1. Run `docker exec -it vuejs sh`
1. Open Terminal (inside VSC) (Ctrl + `)
1. Run `npm install -g @vue/cli`
1. Run `npm install`
1. run `npm run serve`
1. run `cd BE`
1. run `node server.js`
